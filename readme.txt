+++++++++Welcome to my simple Search Engines automation framework+++++++++++++++++

Project title : SearchEngineAutomation

Project installation steps:

1. Download the app and unzip to a folder of your choice
2. Open command line and go to the folder of the application
3. Run the mvn command as follows
   mvn test -Dbrowser=chrome -Dkeyword=Ethiopia or
   mvn test -Dbrowser=firefox -Dkeyword=anyword you want to search
